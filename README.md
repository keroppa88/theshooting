# ■ ゲーム「THE SHOOTING」最終採用仕様まとめ

============================================================

![](Game/playgame.jpg)

### 【1】基本仕様

ブラウザゲーム（HTML5 Canvas）

画面サイズ：480×640

縦スクロール型のシューティング（自機は自由移動）

背景は暗い宇宙＋星ドット

自キャラはドット絵表示（カスタムマップ採用済み）

ゲームパッド（Xbox）/ キーボード両対応

敵を撃破すると音楽開始

### 【2】タイトル画面

タイトルロゴ：THE SHOOTING

PUSH ANY BUTTON（点滅表示）

キー操作：SPACE / ENTER / A / START で開始

画面下に自機ドット絵が飾りとして表示

スタート時点では BGM は再生されない（後述）

### 【3】自機

ドットマップ確定済み（5番が濃いグレーエンジン）

移動速度：5

通常ショット：前方向 1 発

パワーアップ時：前方 3 方向（-20°, 0°, +20°）

### 【4】通常敵

上から降下する赤系機体

スコアに応じて弾を撃つ敵（Shooter）が混ざる

スコア 1300 超えで弾道パターンが「2WAY」に変化

### 【5】中ボス（飛行機）

独自の大型ドット絵（横長）

色指定 8番 → 濃いグレー に変更済み

HP：20

出現条件：300点ごと（300 / 600 / 900 / 1200 /1500）

1500点時のみ左右2体同時に出現

### 【6】ミサイル敵（新規）

スコア 500 / 600 / 700 / 800 / 900 / 1000 到達時に各2発出現

縦長ドット絵（あなたが与えた 1=白 / 2=赤 のマップを採用）

上下反転して実装済み（赤が進行方向）

通常敵の 1.5倍速度

爆破時は通常敵と同じ爆発エフェクト

撃破スコア：20

### 【7】スター（パワーアップアイテム）

スコア 1000 / 1300 で 1 個ずつ出現

自機が触れると消える

効果：5 秒間だけ 3WAY ショット

効果音：star.mp3

### 【8】BGM仕様

ブラウザ規制回避のため「最初の敵を撃破した瞬間」に BGM が鳴り始める

それまでは unlockBgm() により準備状態へ移行

BGM：music.mp3（ループ）

### 【9】最終ボス（モアイ）

ドット絵はあなたが提示したモアイ画像を参考に「リアル感のあるグレー階調」で構築

表示サイズ：元画像の 1/3 × 1.5倍拡大

出現位置：画面上 1/4 の高さ

挙動：左右に往復移動

HP：50

攻撃：口中央から 3方向（0°,±30°）発射

発射間隔：3秒（180フレーム）

### 【10】最終ボス撃破後の演出

#### ■ 改良済み：花火風の多重リング爆発

15重のリングで構成

1つ目が拡大し、後続がわずかに遅れて追随

カラーリング：白→黄→橙→赤→紫→青→緑 などを順に使用

最大半径：キャンバスの 0.4 から 1.5倍へ拡張（あなたの指示）

画面中心ではなく ボスの位置から展開

中心部は途中から「白→赤白点滅」

火花エフェクトは使わない（あなたの判断で削除）

#### ■ 爆発終了直後

画面が 白く一瞬フラッシュ（点滅）
→ クリア演出へ移行

【11】クリア演出（エンディング）

星空を背景に自機がゆっくり上昇（5秒）

その後、last.jpg が画面全体に表示される

「CONGRATULATION!」

ボタンでタイトルへ戻る

### 【12】その他システム

当たり判定は矩形ベース

敵・ミサイル・自弾・敵弾すべて削除処理あり

爆発エフェクトは通常敵用に4種類ランダム採用

Gamepad（Xbox）入力に対応：

移動：スティック / 十字キー

ショット：Aボタン

スタート：STARTボタン


ゲームオーバー：Rキーで即リスタート
